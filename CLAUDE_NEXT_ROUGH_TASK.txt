任务 A：新增一个“最终结论”聚合层
后端新增 backend/final_decision.py，提供函数：

build_final_decision(fund, risk, forecast, prediction, decision_advice, position=None) -> dict

返回结构建议：
{
  "headline": "当前偏观察，不适合追买",
  "direction": "up" | "down" | "neutral" | "uncertain",
  "direction_label": "偏涨" | "偏跌" | "震荡" | "不确定",
  "action": "buy_watch" | "hold" | "reduce" | "avoid" | "observe",
  "action_label": "可小额关注" | "继续持有" | "降低仓位" | "暂不参与" | "观察",
  "confidence": "高" | "中" | "低",
  "up_probability_7d": number | null,
  "up_probability_30d": number | null,
  "risk_score": number | null,
  "why": string[],
  "watch": string[],
  "warning": string[],
  "disclaimer": "仅供个人研究参考，不构成投资建议。"
}

聚合规则要保守、简单：
- 如果 prediction 7d/30d 都是低置信或 has_positive_edge 为 false：direction = "uncertain"，confidence 最多“低”，why 加“历史验证未显示稳定预测优势”。
- 如果 7d 和 30d 都偏涨，且至少一个 confidence 为“中”或“高”，risk_score < 60：direction = "up"。
- 如果 7d 或 30d 偏不涨/偏跌，且 risk_score >= 60：direction = "down" 或 "uncertain"，action 偏 observe/reduce。
- 如果有持仓 position，且 risk_score >= 70：action 至少 reduce/observe，不要输出积极买入。
- 如果没有持仓，且风险高或预测无优势：action = observe 或 avoid。
- headline 必须是一句话，直接说结论，不超过 25 个汉字。
